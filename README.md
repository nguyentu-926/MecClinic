# MecDental

## Mục tiêu
# Phát triển một website hỗ trợ bệnh nhân đặt lịch khám trực tuyến và giúp phòng khám quản lý hoạt động hiệu quả. Hệ thống giảm tải thủ tục giấy tờ, tối ưu quy trình đặt lịch, quản lý hồ sơ bệnh nhân và tra cứu kết quả khám.

 ## Đối tượng sử dụng

Bệnh nhân: đặt lịch hẹn, theo dõi lịch khám, nhận thông báo nhắc lịch.

Bác sĩ: quản lý lịch hẹn, xem hồ sơ bệnh nhân, cập nhật kết quả khám.

Nhân viên phòng khám: quản lý đăng ký khám, điều phối phòng khám.

Quản trị viên: giám sát toàn hệ thống, quản lý tài khoản, báo cáo thống kê.

## Sản phẩm mong đợi

Website cho phép bệnh nhân đăng ký tài khoản, đặt lịch khám trực tuyến.

Quản lý hồ sơ bệnh nhân, lịch hẹn, và thông tin bác sĩ.

Tính năng nhắc lịch qua email.

Trang quản trị cho phòng khám theo dõi báo cáo: số ca khám, tỉ lệ đặt lịch, tình trạng phòng.
